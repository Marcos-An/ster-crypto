## Getting Started

### Installation and Running Locally
 
1. Clone the repo
   ```bash
   
   git clone https://github.com/Marcos-An/ster-crypto.git
   
   ```
2. Install node modules with yarn/npm packages
   ```bash
   
   yarn || npm install 
   
   ```
   
3. Run with yarn/npm packages
   ```bash
   
   yarn dev || npm run dev 
   
   ```
